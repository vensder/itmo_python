2. Сохранить объект в файл можно так:
»> import pickle
»> data = {
... 'a': [1, 2.0, 3, 4+6j],
... 'b': ("character string", b"byte string"),
... 'c': {None, True, False}
... }
»>
»> with open('data.pickle', 'wb') as f:
... pickle.dump(data, f)
...

Тогда так его можно прочитать:

»> with open('data.pickle', 'rb') as f:
... data_new = pickle.load(f)

Сделайте простую базу данных. При первом запуске программы, она создает файл pickle с данными. При последующих открывает этот файл и выводит на экран содержимое.
